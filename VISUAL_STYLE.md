# Tradewinds Learn Visual Style

## Core Concept: Refraction & Lenses

Capturing "Multiple Paths," "No Speed Limits," and Hawaiian complexity. Makawalu means seeing with eight eyes/multiple perspectives. This direction uses optics and light to show that Tradewinds students see the world differently.

**Visual Metaphor:** Refraction and Lenses

**Why it works:** It visualizes the "extraordinary mind." It suggests depth and complexity, reinforcing that these students connect ideas others miss.

## Graphic Elements

- Subtle prismatic effects, overlapping transparent layers, and light spectrums
- Subtle rainbow refraction patterns and soft shadows in subtle but delightful ways
- This is optics (physics), not "rainbows" (which feels preschool)

## Photography

- Candid shots taken through glass, reflections, or using depth-of-field
- Isolate a single sharp detail while the rest is blurred
- Documentary style, not posed

## Typography

- Modern Serif fonts that feel academic but contemporary (like Tiempos or Editorial New)

## Color Palette

| Name | CMYK | Usage |
|------|------|-------|
| Spectrum Indigo | C:95 M:100 Y:20 K:10 | Primary dark / navy backgrounds |
| Prism Violet | C:60 M:90 Y:10 K:0 | Accent / interactive elements |
| Refraction Cyan | C:80 M:20 Y:10 K:0 | Secondary accent |
| Light Flare Yellow | C:5 M:10 Y:90 K:0 | Highlight / attention |
| Optic White | C:1 M:1 Y:1 K:0 | Backgrounds / clean space |

## Application to Tradewinds Learn UI

- Use prismatic/refraction effects subtly in gradients and hover states
- The persona avatar circle can incorporate subtle light-spectrum edges
- Chat header and sidebar gradients should echo the Spectrum Indigo → Prism Violet gradient
- Perspective lens icons reinforce the "seeing differently" metaphor
- Keep effects restrained — sophistication over spectacle
